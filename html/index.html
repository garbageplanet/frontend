Activate a tile



This tool is not yet available.






Tiles are used to overlay a summary of map features. They can serve as feeds for a given perimeter and which can be accessed via our API. They can also be used to restrict actions on an area during games. Specify a tilename and it can be called through our API using @tilename. If a name is not set, a random id will be assigned to the tile. Drag the sides of the polygon on the map to set the tile's area or click on the line to add more handles to reshape the tile.


Mark marine litter


This tool is not yet available.

On a scale from 1 to 10, amount of garbage:












We use wind and coastal configuration data to identify regions where marine litter accumulates on the shores. Our algoritms are, however, not as good as human eyes and you can use this tool to mark area on the map where you know there is always a lot of trash on the shorelines. Coastal highlights are visible and accessible between zoom levels 5 and 20 but you can only draw them at zoom level 17 or more (zoom to level 17 now). Click on the polyline icon on the left to manually draw a highligthing line on the map. You can cancel drawing at any time by pressing Esc. Alternatively, you can upload a GeoJSON polyline. You can even link to a WMS resource in order to create the highlight.
