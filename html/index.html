Organize a cleaning event.








A marker for a cleaning event is meant for a location where trash has already been identified. Users can choose whether to create a Facebook event simultaneously. Login or register with an email to create an event.

Anybody can anonymously mark trash on the map. Markers can only be placed between zoom level 15 and 20 (zoom to level 15 now). When editing any of the marker type, all the changes are tracked and publicly visible in the marker information panel.

Create an area





This tool is not yet available.






Areas are used to overlay a summary of map features. They can serve as feeds for a given perimeter and which can be accessed via our API. They can also be used to restrict actions to a zone area during games. Specify an areaname and it can be called through our API using @areaname. If a name is not set, a random id will be assigned.

Paths identify long stretches of land that accumulate garbage, such as shorelines and road shoulders. Click the Start drawing button in the first tab of this form to start drawing. You can cancel drawing at any time by pressing Esc. Alternatively, you can upload a GeoJSON polyline or link to a WMS resource in order to display the path.

Privacy policy

As a platform that allows users to submit sensitive information related to environmental pollution, we make certain that none of the information we collectthat could potentially identify a user are disclosed to any third party. Furthermore, the user retains the copyright to any uploaded file or image.
